# AI-Based Crowd Behavior Analysis

## 👩‍💻 Modified by Mounika

This project detects abnormal crowd behavior using AI techniques to improve tourist safety.

## 🚀 Features
- Real-time crowd monitoring
- Abnormal activity detection
- Smart alert system
- Backend + frontend integration

## ⚙️ Tech Stack
- Node.js
- Express.js
- React (Admin Dashboard)
- Firebase

## ▶️ How to Run

### Backend
cd backend  
npm install  
node server.js  

### Frontend
cd frontend/admin-frontend  
npm install  
npm run dev
