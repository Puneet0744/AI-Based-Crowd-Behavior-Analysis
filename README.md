# 🌍 AI-Based Crowd Behavior Analysis & Smart Tourist Safety System

## 📌 Overview

This project is an intelligent system designed to monitor, analyze, and improve **tourist safety** using AI-based crowd behavior analysis.
It combines real-time tracking, incident detection, and smart alerts to ensure safer travel environments.

---

## 🚀 Features

* 📍 Real-time location tracking
* 🧠 AI-based crowd behavior analysis
* 🚨 Incident detection & emergency alerts (SOS)
* 🗺️ Geo-fencing for restricted/safe zones
* 📊 Admin dashboard for monitoring
* 📱 Mobile app for tourists

---

## ⚙️ Installation & Setup

### 🔹 Clone the repository

```
git clone https://github.com/Puneet0744/AI-Based-Crowd-Behavior-Analysis.git
cd AI-Based-Crowd-Behavior-Analysis
```

---

### 🔹 Backend setup

```
cd backend
npm install
npm start
```

---

### 🔹 Admin Dashboard

```
cd admin-dashboard/frontend
npm install
npm start
```

---

### 🔹 Mobile App

```
cd mobile-app
npm install
npx expo start
```

---

## 🔐 Environment Variables

Create a `.env` file in the backend folder and add:

```
PORT=5000
FIREBASE_API_KEY=your_key
```
