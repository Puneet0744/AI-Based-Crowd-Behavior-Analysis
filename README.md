# YOLO People Counting System

This project detects and counts people in a video using YOLOv8.

## Features
- Detects people in every frame
- Draws bounding boxes around people
- Updates people count every 5 seconds

## Technologies Used
- Python
- YOLOv8
- OpenCV
- Supervision

## Installation

Install required libraries:

pip install -r requirements.txt

## Run the Program

python people_count.py

## Example Output

The system shows:
- Bounding boxes around people
- Total people count updated every 5 seconds
